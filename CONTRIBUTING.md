# Contributing to Seattle Parking Weather Analysis

Thank you for your interest in contributing! This project is open to everyone.

## Getting Started

1. Fork the repository
2. Clone your fork:
   ```
   git clone https://github.com/YOUR_USERNAME/seattle-parking-weather-analysis.git
   ```
3. Create a virtual environment and install dependencies:
   ```
   python -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   ```
4. Create a new branch for your feature:
   ```
   git checkout -b your-feature-name
   ```

## How to Contribute

- Pick an open issue from the Issues tab
- Comment on the issue to let others know you're working on it
- Submit a pull request when ready — include a short description of what you changed and why

## Good First Issues

If you're new to the project, look for issues tagged `good first issue`. These are beginner-friendly tasks.

## Code Style

- Use clear variable names
- Add a short comment if the logic isn't obvious
- Keep functions small and focused

## Questions?

Open a GitHub Discussion or comment on an issue.
